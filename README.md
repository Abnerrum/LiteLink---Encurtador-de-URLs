🔗 LiteLink - Encurtador de URLs
Projeto desenvolvido para colocar em prática conceitos de rotas dinâmicas, manipulação de banco de dados e contagem de acessos com Laravel.
🛠️ Tecnologias
PHP 8.2+
Laravel
Tailwind CSS
SQLite
🚀 Como rodar
git clone [url-do-repositorio]
composer install
cp .env.example .env
php artisan migrate
php artisan serve
